# 技术学习追踪系统

> 使用 AI 辅导和苏格拉底教学法进行深度技术学习

## 📚 简介

这是一个基于 AI 辅导的技术学习追踪系统，通过苏格拉底教学法和系统化的进度追踪，帮助你掌握任何技术知识。

**核心特点：**
- 🧠 苏格拉底教学法 - 主动思考，深度理解
- 📝 会话记录 - 详细记录每次学习过程
- 📊 进度追踪 - 可视化学习进度和知识缺口
- 🎯 个性化学习 - 基于现有知识构建，避免重复
- 🔄 持续改进 - 识别缺口，有针对性地巩固

## 🎯 适用场景

适合学习任何技术领域：
- 后端开发（Java, Go, Spring, 微服务等）
- 前端开发（React, Vue, TypeScript 等）
- DevOps/基础设施（Docker, Kubernetes, CI/CD 等）
- 架构和设计（系统设计, DDD, 设计模式等）
- 工具和实践（Git, 测试, 代码审查等）
- 新兴技术（AI/ML, Web3, Serverless 等）

## 🚀 快速开始

### 1. 初始化学习环境

```bash
# 1. 复制模板文件
cp SESSION-TEMPLATE.md sessions/SESSION-TEMPLATE.md
cp STUDY-TRACKER-TEMPLATE.md progress/study-tracker.md

# 2. 更新 progress/study-tracker.md
# 填写你的学习主题、开始日期等信息
```

### 2. 开始学习

1. 打开 AI 助手（如 Claude Code）
2. 告诉 AI 你想学习的主题
3. AI 会通过苏格拉底教学法引导你学习
4. 每次学习后，AI 会自动记录会话和更新进度

### 3. 追踪进度

- 查看当前进度：`progress/study-tracker.md`
- 查看学习历史：`sessions/YYYY-MM-DD/session-notes.md`
- 识别知识缺口：在 `study-tracker.md` 中查看缺口部分

## 📁 项目结构

```
/sessions/                      # 学习会话记录（动态创建）
  /YYYY-MM-DD/                 # 每日会话文件夹
    session-notes.md           # 详细会话记录
  SESSION-TEMPLATE.md          # 会话记录模板
/progress/                      # 学习进度追踪
  study-tracker.md            # 综合进度追踪器（从模板创建）
  STUDY-TRACKER-TEMPLATE.md   # 进度追踪模板
AGENTS.md                      # AI 助手工作指南
README.md                      # 本文件
```

## 🎓 学习方法

### 苏格拉底教学法（4 步骤）

1. **初步探索**：AI 询问你"对这个主题已经了解了什么？"
2. **精炼解释**：基于你的理解，AI 提供约 200 字的简洁解释
3. **理解验证**：AI 提出后续问题检查你是否真正理解
4. **适应性跟进**：根据你的回答调整教学方式

### 会话记录

每次学习后，系统会自动记录：
- 你提出的问题和初步理解
- 解释的概念和教学方法
- 你对理解检查的回应
- 识别的知识缺口
- 掌握的主题及自信程度
- 完成的练习和代码示例
- 后续学习计划

### 进度追踪

`study-tracker.md` 追踪：
- 整体学习进度和统计
- 按领域分类的主题掌握情况
- 知识缺口（高/中/低优先级）
- 学习计划和目标
- 资源清单和复习计划

## 💡 使用建议

### 学习新主题

1. **从基础开始**：先理解核心概念和原理
2. **实践验证**：通过代码示例和小项目应用知识
3. **深入探索**：研究源码和底层实现
4. **扩展学习**：学习相关技术和生态系统

### 复习策略

- **定期复习**：基于知识缺口优先级安排复习
- **实践验证**：通过项目验证理解程度
- **关联学习**：将新旧知识建立联系

### 遇到困难

- 不要担心知识缺口 - 这是学习的一部分
- 系统会自动识别和追踪缺口
- 后续学习会针对性地填补缺口

## 🛠️ 自定义

### 修改学习领域

编辑 `progress/study-tracker.md`，调整：
- 领域/分类
- 主题列表
- 优先级权重

### 调整学习计划

根据你的情况修改：
- 学习频率
- 目标设定
- 时间规划
